# AI 宠物面相 - 产品说明

## 定位

面向猫狗主人，提供轻量、可分享的宠物娱乐报告。

## 用户流程

1. 进入首页
2. 上传宠物照片
3. 选择宠物类型并填写名字
4. 生成免费报告
5. 模拟解锁完整版

## 报告模块

### 免费版

- 宠物标签
- 一句话总结
- 性格分析
- 它怎么看主人
- 分享文案

### 完整版

- 深度性格
- 福气运势
- 和主人的关系
- 健康观察提示
- 幸运物 / 幸运零食
- 专属点评

## 当前实现

- 图片上传到本地 `server/uploads/`
- 报告持久化到 `server/data/reports.json`
- 支持 `GET /api/reports/:id`
- 支持 `POST /api/reports/:id/unlock`

## 后续可升级

- 接入真实对象存储
- 接入视觉模型
- 接入微信支付
- 增加海报生成
